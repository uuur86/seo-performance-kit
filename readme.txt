== Description ==

The CodePlusDev SEO and Performance Kit is a WordPress plugin designed to improve your website's SEO and overall performance. 

Features include:
* Automatically adds 'display=swap' to Google Font links
* Adds 'link rel="preload" as="font"' attributes to font files
* Easy to use, just install and activate

== Installation ==

1. Upload `codeplusdev-seo-performance-kit.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. No configuration needed, the plugin works out of the box

== Frequently Asked Questions ==

= Do I need to configure anything? =

No, the plugin works out-of-the-box. Just install and activate.

= Is it compatible with other SEO plugins? =

Yes, it should work fine with most SEO plugins, but it hasn't been tested with all of them.
